# gbemulator
Game Boy Emulator - Not designed to be cycle accurate.

## A note on ROMs

This project uses only publicly available ROMs for testing but they will be excluded from the github repo. If you intented to use this software please only use legally acquired ROMs / ROMs made from games you own / Homebrew ROMs you legally have access too.
